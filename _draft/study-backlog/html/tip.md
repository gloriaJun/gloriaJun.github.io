#### viewport
viewport를 설정하면 다양한 모바일 기기에서도 페이지의 너비나 화면 배율을 설정할 수 있습니다.

`head` 태그 사이에 아래의 코드를 추가하여 준다.   
기본적으로 데스크탑 브라우저에서는 viewport 메타 태그를 사용하지 않으므로 무시된다.
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
